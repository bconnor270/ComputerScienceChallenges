Introduction

You can check out the blog post I wrote about this process here.

This guide is designed to take you from start to finish through the process of running a virtual machine with QEMU and setting up networking. Other documentation on the web, especially regarding wireless networking with QEMU, is quite scattered so I hope this will help.

This walkthrough is based on using Ubuntu 18.04; however the process is similar on other Linux distributions.

This tutorial is designed to be followed by someone who has no prior experience with virtual machines; however some familiarity with the Linux command line is helpful as the vast majority of this process is carried out from within it.

Some familiarity with basic networking and common protocols is helpful if you intend to attempt wireless networking.

Lingo

A key for some common phrases throughout this guide:

- "Host machine" / "Host" : This is your computer; the machine doing the virtualizing.

- "Guest machine" / "Guest" : The machine you will run with QEMU; the machine being virtualized.

- "Terminal" : The ordinary Linux command line.

- "VM" : Shorthand for Virtual Machine

- "KVM": Kernel Virtual Machine; in short, a technology built into the Linux kernel that allows it to run multiple, isolated virtual machines. More info here.

A final note: there will be some information within this guide that although not compulsory for the walkthrough, will hopefully deepen your understanding of what's going on. Feel free to ignore it!
