User-Mode Networking

Intro

User-mode networking is the easiest kind of networking to set up. We simply have to declare that we want the user VLAN mode and specify a NIC at the time of launching our virtual machine. See an example below: 

    sudo qemu-system-x86_64 -m 3072 -enable-kvm -hda test-image.img -netdev user,id=net0 -device e1000,netdev=net0,
    mac=DE:AD:BE:EF:12:34

'-netdev user,id=net0' : Here we declare that we want a user-mode VLAN, and we name it net0.

'-device e1000,netdev=net0,mac=...' : Here we are specifying that we want the e1000 NIC, and attribute it to our VLAN and then assign it a MAC address.

This will launch your VM with User-Mode networking.
