Install QEMU

QEMU and its dependencies are packaged within most Linux distributions. Open a terminal window and type the command: 

	sudo apt-get install qemu qemu-kvm libvirt-bin

You will be prompted for your password and then your machine will install QEMU and its dependencies.

The keyword 'sudo' in this command stands for Super User DO. It allows commands to be run as the root user without actually having to be signed in as the root user.

Background Knowledge : the 'apt-get' part of this command stands for Advanced Package Tool - an extremely common tool used to manage the installation of software onto Debian/Ubuntu distributions.
